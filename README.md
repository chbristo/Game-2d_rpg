
Φάκελοι:
    inlude: τα .h αρχεια
    main: το αρχείο main και το Μakefile αρχείο
    modules: βοηθητικές συναρτήσεις και συναρτήσεις κλάσεων

Εντολές εκτέλεσης:
    Μέσα στο φάκελο main που βρίσκεται το Makefile αρχείο
    make clean: διαγραφή .o αρχείων
    make main: compile όλων των αρχείων
    make run:  εκτέλεση όλων των αρχείων
    make clean main run: προτινώμενη εντολή εκτέλεσης
